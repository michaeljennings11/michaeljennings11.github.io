Tips For New Researchers

THIS PAGE IS UNDER CONSTRUCTION.

Transitioning from lecture based academic courses into research for the first time can be difficult for some. The learning curve is quite steep for most students because the skills that facilitate successful research are not explicitly taught in undergraduate courses. Here I suggest tips and skills I have picked up along the way of starting Astrophysics research that I wish I knew about when first starting out.

Reference Organizer

Lecture based academic courses taken as an undergraduate usually use no more than one or two textbooks as references for the material. When first starting research, students have a lot of "catching up" to do in regards to familiarizing themselves with the subfield of interest. It is common to be given the first task of reading classic and recent literature when starting a new project with a research mentor. As time goes on, the amount of papers to read or to reference in your publication can become very large. I've noticed beginner students tend to download the pdf's of the neccessary papers and have all the files stored in a directory on their computer. Reference organization in this manner becomes unwieldy fairly quickly for these reasons: (1) you cannot easily search for keywords across the whole directory; (2) organizing papers by different projects is more difficult; (3) there are no convenient tools for creating citations for a project paper.

For these reasons I recommend Mendeley to organize all references and to easily create citations for papers.

Text Editor

I recommend vim as a text editor.

Version Control

I recommend git with github for a version control system.
